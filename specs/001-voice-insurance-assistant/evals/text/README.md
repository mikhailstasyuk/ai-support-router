# 001-voice-insurance-assistant/evals/text

This feature defines a text-only evaluation specification pipeline.

## Scope

- Includes deterministic text scenario definitions, requirement mappings, result schemas, and run metrics.
- Excludes voice execution testing.

## Voice Compatibility Boundary

- Existing voice requirement IDs must remain unchanged.
- Text-eval artifacts may reference voice requirements only as compatibility links.
- Text-eval artifacts must not introduce executable voice test steps.
